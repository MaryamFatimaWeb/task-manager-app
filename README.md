# 📋 Task Manager App

A Full Stack Task Management application built with **React.js**, **Node.js**, **Express.js**, and **MongoDB**. The application allows users to manage daily tasks efficiently with complete CRUD functionality.

---

## 🚀 Features

- ➕ Add New Tasks
- 📋 View All Tasks
- 🗑️ Delete Tasks
- 💾 MongoDB Database Integration
- ⚡ REST API
- 📱 Responsive User Interface

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Axios
- CSS

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/MaryamFatimaWeb/task-manager-app.git
```

### Backend

```bash
cd server
npm install
npm run dev
```

### Frontend

```bash
cd ../client
npm install
npm start
```

---

## 🌐 Environment Variables

Create a `.env` file inside the server folder.

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
```
